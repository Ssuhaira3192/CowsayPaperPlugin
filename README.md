# CowsayPlugin 🐮

CowsayPlugin is a lightweight Paper Minecraft plugin that recreates the classic Linux `cowsay` command inside the game. It allows players to display custom messages in a speech bubble above an ASCII cow.

## Overview

This plugin adds a simple `/cowsay` command. When a player runs the command, their message is displayed in a speech bubble followed by an ASCII cow.

## Features

- `/cowsay <message>` command
- Dynamic ASCII speech bubble
- Classic cow ASCII art output
- Lightweight and fast
- No dependencies

## Requirements

- Paper server
- Minecraft 26.1.2
- Java 17 or higher

## Installation

Download the latest `.jar` from Releases, place it in your `plugins` folder, and restart your server.

## Commands

### `/cowsay <message>`

Displays a message in a speech bubble above a cow.

Example:
/cowsay hello world

Output:

 _______________
< hello world >
 ---------------
        \   ^__^
         \  (oo)\_______
            (__)\       )\/\
                ||----w |
                ||     ||

### About

CowsayPlugin is inspired by the Linux cowsay utility and brings it into Minecraft as a fun command.
